# Project Overview
Analysis of bank customer data to identify churn drivers and high-risk customer segments using Tableau.

# Objectives

- Improve retention strategies  

# Tools
- Tableau  
- SQL  
- Excel

# Dataset

# Key Insights
- Low engagement increases churn  
- Certain age groups churn more  
- Fewer products lead to higher exit  

# Dashboard
Interactive Tableau dashboard with key filters.

# Conclusion
Supports data-driven decisions to reduce churn and improve customer experience.
